# 🧩 PPS-25-PrologForDummies

**PrologForDummies** è un'applicazione desktop progettata per guidare gli utenti nell'apprendimento del linguaggio **Prolog**, dai concetti base alle logiche più avanzate.

---

## Caratteristiche Principali
* **Motore Prolog Integrato**: Validazione dinamica delle risposte anche con uso di tuProlog.
* **Apprendimento Progressivo**: Livelli strutturati per coprire dai fatti di base alle regole complesse.
* **Sessioni Persistenti**: Salvataggio dei progressi utente.
* **Interfaccia Moderna**: GUI intuitiva sviluppata in ScalaFX.
* **Due tipologie di Quiz**: Possibilità di rispondere a quiz a risposta multipla e Fill in the blank.

## Tech Stack

* **Language**: Scala 3
* **Logic Engine**: tuProlog
* **UI Framework**: ScalaFX (JavaFX)
* **Build Tool**: sbt
* **Testing**: ScalaTest
